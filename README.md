# 38-2nd-WeFrip-backend
박은송, 신인혁, 천송인

- 프로젝트 사이트 -> **Frip**
- 나이키는 운동선수들을 위한 스포츠 의류 기업으로 세계적인 스포츠웨어 e-commerce 사이트이다. -> 변경 요청
- 직관적인 UI와 다양한 상품 필터링으로 구성되어있어, 사용자에게 편리한 UX를 제공한다. -> 변경 요청
